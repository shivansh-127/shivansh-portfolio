# Shivansh Saxena — Portfolio

Built with React + Vite + Framer Motion. Dark theme, premium UI.

## Quick Start

```bash
npm install
npm run dev
```

Open http://localhost:5173

## Build

```bash
npm run build
```

## Deploy to Vercel

1. Push to GitHub
2. Import repo on vercel.com
3. Vercel auto-detects Vite — hit Deploy

No env vars needed.
